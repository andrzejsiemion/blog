The gear icon at the top-right of the home screen opens Settings. Tap **Done** to close.

| Section | What it controls |
|---|---|
| **Sort Cards By** | Order of the home grid |
| **Barcode Display** | Toggles the brightness boost and lets you lower the boost level to save battery |
| **Appearance** | System / Light / Dark theme |
| **Language** | English / Polish — Scandio asks you to restart when you change this |
| **Backup** | Duplicate-import policy + Export / Import buttons |
| **About Scandio** | App version, developer info, privacy policy, source code, license |

![Settings page showing sort, brightness, appearance, language, backup sections](/scandio/manual/10-settings.png)

## About Scandio

Tap **About Scandio** to see the version, the developer's website and GitHub, the [privacy policy](/scandio/privacy/), the MIT license, and the open-source project URL.
